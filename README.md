# Sugar-Rush
It's a straightforward and small indie game.
Get ready for a Sugar Rush! Play as a wobbly jelly racing through a sticky-sweet landscape. Avoid gooey slime monsters, and bounce your way to the finish line in this fast-paced survival game.
The Game can be accessed from the Unity website and this link: "[https://play.unity.com/en/games/cd3949f7-8fc6-49a9-92ae-c27045d237a6/sugar-rush](https://play.unity.com/en/games/cd3949f7-8fc6-49a9-92ae-c27045d237a6/sugar-rush)"
